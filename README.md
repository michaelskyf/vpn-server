# VPN server
## _Simple VPN server for GNU/Linux written in C++_

## About
The project was written during an internship at Apriorit in summer 2022

## Features
- Traffic encrypted via TLS
- Simple

## Building
```
cmake .
make
./bin/vpn-server --help
```

## Supported compilers
- gcc
- clang

## Supported Operating Systems
- GNU/Linux
